# Running

On Linux:

Install dependencies:

```
sudo apt install libsdl2-dev libsdl2-image-dev libsdl2-mixer-dev libsdl2-ttf-dev
```

Run:

```
sh launch_game.sh
```

On Windows:

```
launch_game.bat
```

On MacOs:

The Mac build is a WIP.
